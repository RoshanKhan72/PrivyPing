
### 🔄 What’s New in v1.1.2
- Integrated **real-time notification analysis**
- Integrated **offline multimedia (image & video) analysis**
- Removed dependency on external APIs
- Improved modular architecture for scalability

---

## 🛠️ Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Database:** Room
- **System Services:** NotificationListenerService
- **Architecture:** Modular / MVVM-style separation
- **AI/ML:** Custom on-device models
- **Version Control:** Git (branch-isolated releases)

---

## 🔒 Privacy Principles

- ❌ No internet usage
- ❌ No cloud APIs
- ❌ No user data collection
- ❌ No external servers
- ✅ Fully offline
- ✅ On-device processing only

---

## 🧪 Development Notes

- Experimental features are always isolated in separate branches
- Stable releases are never force-modified
- Room DB schema changes may require a one-time app reinstall during development
